# SPI pins configuration

| Pin Name      |Signal On Pin  | 
| ------------- |:-------------:|
| PC10          | SPI3_SCK      |
| PC11          | SPI3_MISO     |
| PD6           | SPI_MOSI      | 
| PD2           | SPI_CS as GPIO Out      |
